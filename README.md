# dress-experiments

Experiments, benchmarks, and downstream applications for the
[DRESS graph framework](https://github.com/velicast/dress-graph).

## [Isomorphism](isomorphism/)

Delta^k-DRESS on hard graph-isomorphism benchmarks: 51,820 graphs
across 34 families (16 SRG parameter sets + 18 constructed families),
achieving 100% within-family separation on 576,107,835
non-isomorphic pairs. Includes the
[CFI staircase](isomorphism/#cfi-staircase): Delta^k-DRESS empirically
matches the (k+2)-WL boundary on tested CFI instances (k = 0..3).
